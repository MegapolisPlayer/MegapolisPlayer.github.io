Introducing Versailles

The extensive 2D C++ framework with everything a developer needs.

Simple and Fast

Versailles has many functions to get you up and running in a few minutes at most, while keeping performance suitable for any medium-to-high performant application - be it a game or a serious piece of software.

Numerous easy-to-use abstractions

From complex multithreading through events to batch rendering, Versailles has abstractions for you at every step of the way to make developing easier and more fun.

Every error, logged

Versailles features 2 complementary error handling systems (one extra planned) to catch most (if not all) or the errors that might occur and the object that is causing them.

Not just games

Versailles can be used to develop software far beyond just games - from tools to visualizations, Versailles gets you covered.

Free to use

Versailles is completely free-to-use unless it is for a commerical purpose - then we take a 10% cut.

Info

  • Versailles is made with C++ in mind.
  • Versailles runs only on 64-bit hardware due to the hardware limitations and the market share of 32-bit machines.
  • It is recommended to always use the newest version of Versailles as it is the only one to recieve security updates. (The previous version will recieve security-only updates only if the newer version contains a major API change which breaks compatibility.)

Platforms
