Summer redesign (Changelog for 2023.1)

I redesigned this website in Summer 2023 to a) not look like trash and b) so I can actually use it for marketing, devlogs and so on.
The key points of this redesign were:

  • Refresh graphics to the modern standard
  • Better mobile UI
  • Easier navigation

  • A vastly improved bug reporting system
  • A more consistent color scheme
  • Better accessibility
  • Improve CI/CD system (in Github, no CMS yet)
All of the following was successfully implemented in approx. 5 days. Most of the website was rewritten from scratch however you can still see the remnants of the old website (for example the "sections").
