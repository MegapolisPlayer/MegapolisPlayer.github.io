About Me

I'm a young student who REALLY likes C and C++. These languages give you a lot of control to make the application look and feel EXACTLY as you want. And it also doesn't force you to use OOP (like Java or C#) which is, in my opinion, really good. I think Bjarne Stroustrup (the orignal creator of C++) puts it best: C makes it easy to shoot yourself in the foot; C++ makes it harder, but when you do it blows your entire leg off.

My magna opera (=lat. for "greatest works")

  • MuffleSpeech - a string changer - discontinued, but still REALLY fun to use
  • CountryFramework - a closed-source cross-platform GUI framework
  • ComInPlat - a yet unfinished common interpreter platform for custom languages (something like LLVM, but worse and for interpreters)
